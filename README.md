# Diablo: A Distributed Analytical Blockchain Benchmark Framework

Diablo, an analytical blockchain benchmark framework focusing on distributed
clients performing real-world workloads.


## Workloads

* ICO
* Auction
* Gameplay
* Gambling
* Wiki/Website Access
* Shipping Containers
* Exchange
